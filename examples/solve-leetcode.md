# Solve a LeetCode Problem

**Runtime:** claude · **RAM:** 1024 MB · **Expected duration:** 30–60 s

Demonstrates: constrained code generation with verification, minimal VM (no browser/node needed).

## Prompt

```prompt
Solve this problem in Python, then verify your solution against the sample cases
before writing the final answer:

"Given an integer array nums, return the length of the longest strictly
increasing subsequence. Example: [10,9,2,5,3,7,101,18] → 4."

Save the solution to /root/output/solution.py with runnable sample inputs at
the bottom. Save /root/output/analysis.md with the time complexity, space
complexity, and why you chose the approach you did.
```
